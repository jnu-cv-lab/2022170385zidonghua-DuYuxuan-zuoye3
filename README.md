# 计算机视觉实验四：图像缩小、恢复与频域分析 (Lab 04)

## 📝 1. 实验目的与背景
本实验是计算机视觉课程的第四次作业。核心目标在于：
1. **空间域操作**：掌握图像的下采样（缩小）原理，以及如何通过不同的空间插值算法（最近邻、双线性、双三次）将图像恢复至原始尺寸。
2. **抗混叠滤波**：对比直接下采样与“先高斯平滑再下采样”的效果差异。
3. **误差评估**：引入均方误差 (MSE) 和峰值信噪比 (PSNR) 作为客观评价指标，量化不同恢复算法的优劣。
4. **频域分析 (硬核核心)**：利用二维离散傅里叶变换 (FFT) 和离散余弦变换 (DCT)，从频率的视角深入分析不同插值算法对图像高低频信号的影响。

## 🎯 3. 核心任务与技术实现
本实验的所有逻辑均在 `main.py` 中实现，自动将处理结果输出至 `build/` 目录。

* **任务 1 & 2：图像预处理与下采样**
  - 读取测试海浪图，并强制转换为单通道灰度图。
  - 分别采用 **直接下采样** 与 **高斯平滑后下采样 (抗混叠)** 两种策略将原图缩小为 1/2。
* **任务 3 & 4：图像恢复与客观评价 (MSE & PSNR)**
  - 使用三种插值法将缩小后的图像放大回原尺寸。
  - **评估结论**：通过计算发现，**双线性内插**的 PSNR 最高（约 21.49 dB），MSE 最小，视觉还原效果最好；而最近邻内插 PSNR 最低（约 18.69 dB），呈现明显的马赛克边缘。
* **任务 5：傅里叶变换 (FFT) 分析**
  - 分别对原图、缩小图和恢复图进行 FFT，并将零频分量移至中心进行可视化。
  - **频域差异分析**：双线性恢复图的频谱在超高频区域比原图暗淡，说明平滑插值无法找回丢失的高频细节。同时，恢复图的频谱中出现了明显的水平与垂直“十字亮线”，这是内插算法在像素网格间填充数据时引入的人为网格边缘效应。
* **任务 6：离散余弦变换 (DCT) 分析**
  - 对图像进行 DCT 变换并统计左上角（宽高的 1/8）低频区域的能量占比。
  - **能量分布解释**：原图与最近邻恢复图的低频能量占比约为 96.5%，而双线性恢复图的低频能量占比激增至 98.19%。这是因为双线性内插等效于低通滤波器，削弱了高频细节，从而大幅提升了低频能量的相对比重。
